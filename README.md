# CTF工具箱

基于 Flask 的离线 CTF 工具箱，通过浏览器访问即可使用，覆盖逆向、密码、隐写、Web、渗透、取证、流量、防火墙、数据库等 11 大模块，130+ API 接口。

## 功能模块

### 智能检测
拖入文件自动识别类型并调用对应模块分析，支持 PE/ELF、ZIP/RAR、图片、PCAP、CSV、日志、内存转储等格式。

## 安装

### 环境要求
- Python 3.8+
- Windows / Linux / macOS

### 安装依赖
```bash
pip install -r requirements.txt
```

### 依赖说明
| 包名 | 用途 | 必需 |
|------|------|------|
| flask | Web 框架 | 是 |
| pillow | 图片处理 | 是 |
| pefile | PE 文件分析 | 否，未安装时降级 |
| scapy | pcap 流量分析 | 否，未安装时降级 |
| rarfile | RAR 压缩包 | 否，未安装时降级 |
| pandas | Excel 分析 | 否，未安装时降级 |
| openpyxl | Excel 读取 | 否，未安装时降级 |

## 使用

### 方式一：启动脚本（推荐）
双击 `start.bat`，自动检测环境、安装依赖、启动服务并打开浏览器。

### 方式二：命令行
```bash
python app.py
# 浏览器打开 http://127.0.0.1:5000
```

## 项目结构
```
ctf_saber/
├── app.py                 # Flask 主程序，138 个 API 路由
├── start.bat              # Windows 一键启动脚本
├── requirements.txt       # Python 依赖
├── templates/
│   └── index.html         # 前端页面（单页应用）
├── solvers/               # 功能模块
│   ├── reverse.py         # 逆向工程
│   ├── crypto.py          # 密码学
│   ├── stego.py           # 隐写分析
│   ├── data_sec.py        # 数据安全
│   ├── archive.py         # 压缩包处理
│   ├── traffic.py         # 流量分析
│   ├── log_analysis.py    # 日志分析
│   ├── web.py             # Web 安全
│   ├── pentest.py         # 渗透测试
│   ├── forensics.py       # 电子取证
│   ├── firewall.py        # 防火墙
│   ├── db_security.py     # 数据库安全
│   ├── server_config.py   # 服务器配置
│   ├── network.py         # 网络配置
│   └── common.py          # 通用工具函数
├── wordlists/             # 字典文件
└── uploads/               # 上传文件临时目录
```

## 技术栈
- **后端**: Python + Flask
- **前端**: 原生 HTML/CSS/JavaScript（单页应用，无框架依赖）
- **特性**: 离线可用、自动降级、文件拖拽上传、结果内联显示
